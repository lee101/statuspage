A technical product map from Applied AI NZ

Here is the short map if you want to explore the more technical parts of the Applied AI NZ ecosystem:

  • Text-Generator.io for text, vision, and speech API work.
  • Open Paths for model routing and provider choice.
  • Web Fiddle and Site Sim for browser-based building and simulation.
  • Netwrck, AI Art Generator, and SimplexGen for multimodal creation.
  • Experiment Flow and Evangeler for growth, testing, and partner programs.
  • Bitbank for crypto forecasting and API access.
  • How NZ for technical writing and build notes.

The best next step is usually narrow: pick one workflow that is slow, fragile, or expensive, then make it measurable before adding more automation.
