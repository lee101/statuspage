Games as retention and learning loops

Applied AI NZ also builds browser game loops because games are a clean way to test retention, learning, multiplayer, and creative interaction patterns.

V5 Games and Addicting Word Games collect text adventures, word games, typing games, maths practice, and larger browser game experiments. Individual examples include Word Smashing, reWord Game, Multiplication Master, and Big Multiplayer Chess.

If you run games or learning products, monitor login, matchmaking, save state, leaderboards, content generation, and asset delivery separately. One red component should not make the whole product look broken.
